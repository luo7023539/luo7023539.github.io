正则表达式

JS正则表达式中的基本知识

1. 什么是正则表达式

在常见的字符串检索或替换中，我们需要提供一种模式表示检索工替换的规则
正则表达式使用单个字符串来描述、匹配一系列符合某个句法规则的字符串
判断正则是否能匹配给定参数字符串

2. JS正则中的特殊字符

需要转义的特殊字符

( [ { \ ^ $ | ) ? * + .

特殊字符说明

  1. $ 匹配输入字符串的结尾位置。请使用 \$
  2. ( ) 标记一个子表达式的开始和结束位置,请使用( )
  3. * 匹配前面的子表达式零次或多次,请使用 *
  4. + 匹配前面的子表达式一次或多次,请使用 +
  5. . 匹配除换行符\n之外的任何单字符,请使用 .
  6. [ ] 标记一个中括号表达式的开始,请使用 [ 或 ]
  7. ? 匹配前面的子表达式零次或一次,请使用 \?
  8. \ 将下一个字符标记为或特殊字符,请使用 \
  9. ^ 匹配输入字符串的开始位置,请使用 \^
  10. {} 标记限定符表达式的开始。请使用 {
  11. ¦ 指明两项之间的一个选择,请使用 \¦

3.预定义的特殊字符

字符 描述

  1. \t 制表符
  2. \n 制表符
  3. \r 回车符
  4. \f 换页符
  5. \v 垂直制表符
  6. \0 空字符
  7. \x0B 垂直tab

4. 范围符号

符号 含义 示例

  1. [] 字符范围 [a-z] [0-9] [a-z0-9]
  2. [^] 字符范围以外 [^a-z]
  3. ^ 行首 ^zf
  4. $ 行尾 zf$
  5. \b 零宽单词边界 \bhello
  6. \B 非\b \Bhello

. 预定义类

字符 等同于 含义 示例

  1. . [^\n\r] 任意字符 /…/.test(‘abc’)
  2. \d [0-9] 数字0-9 /\d\d\d/.test(‘123’)
  3. \D [^0-9] 非\d,即不是数字0-9 /\D\D\D/.test(‘abc’)
  4. \w [a-zA-Z0-9] 数字0-9、字母a-z及a-z、下划线 /\w\w/.test(‘a‘)
  5. \W [^a-zA-Z_0-9] 非\W /\W\W/.test(‘@#’)
  6. \s [\t\n\x0B\f\r] 空格、TAB、换页符、换行符 /\sabc/.test(‘ abc’)
  7. \S [^\t\n\x0B\f\r] 非\s /\Sd/.test(‘a d’)

6. 分组

符号 含义 示例

  1. (x) 分组，并记录匹配到的字符串 /(abc)/
  2. \n 表示使用分组符(x)匹配到的字符串 /(abc)\1/.test(‘abcabc’)

  3. (?:x) 仅分组不捕获 /(?:abc)(def)\1/.test(‘abcdefdef’);

    7.重复

    符号 含义 示例

  4. a,a+ 重复次数>=0,重复次数>=1 贪婪算法 /abc.test(‘abccc’)/

  5. a? a+? 同a,a+ 非贪婪算法 /abc*/.test(‘ab’)
  6. a? 出现0次或1次 /a?bc/.test(‘ab’)
  7. a¦b a或者b /a¦b/.test(‘a’)
  8. a{n},a{n,},a{n,m} 重复n次，重复>=n次,重复n<=x<=m /a{2}/.test(‘aa’)

8.前瞻

正则 名称 描述

  1. (?=exp) 正向前瞻 匹配后面是exp的位置
  2. (?!exp) 负向前瞻 匹配后面不是exp的位置

9.标志位

符号 含义

  1. global 匹配所有的情况
  2. ignoreCase 忽略大小写
  3. multiline 跨行匹配

10.RegExp对象方法

方法 含义 示例

  1. compile 可以改变正则的内容 var reg = /abc/;reg.compile(‘def’);reg.test(‘def’);//true
  2. exec 用正则匹配字符串获取匹配结果 /abc/.exec(‘abcdef’); // “abc”
  3. test 如果匹配返回true /ab/.test(‘ab’)
  4. toString 返回正则内容 /ab/.toString()
  5. exec 返回单次匹配的数组结果

11.字符串的正则方法

方法 含义 示例

  1. search 查找正则匹配对应的索引 ‘abcdefabc’.search(/(abc)\1/)
  2. replace 替换 ‘aabbbcc’.replace(/b+?/); //aa1bbcc
  3. match 匹配 ‘aabbbccbbb’.match(/b+/g) //
  4. split 分割字符串 ‘aabbbcc’.split(/b+/); /[“aa”,”cc”]
