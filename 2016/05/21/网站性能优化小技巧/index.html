发表于 2016-05-21

网站性能优化

  • 尽量减少向服务器请求的次数『减少HTTP请求』
  • CSS/JS 文件进行合并
  • ICON图片也进行合并
  • 雪碧图 css sprite
  • 图片延迟加载
  • 数据异步加载
  • 在移动端，若为一个简单的宣传页，可以把css和js写成内嵌式

图片延迟加载：

  1. 对于首屏内容中的图片，首先给对应区域一张默认图片占位，当首屏内容加载完成后，在开始加载真实的图片

  2. 对于其他屏中的图片：也是给一张默认的图片，当滚动条滚动到对应区域的时候，再开始加载真实图片

给图片区域增加默认图片当做背景
img设置display：none
之后将真实图片显示出来
